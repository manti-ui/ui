# Marquee

A continuously scrolling strip of content. It is a static CSS-animation component
— the content is duplicated for a seamless loop, the copy is hidden from
assistive tech, and the strip holds still under reduced motion or
`data-motion="none"`.

<Demo name="marquee/basic" />

## Basic

Drop any content as `children`; it is repeated to fill the loop. `speed` is the
number of seconds for one full pass, so a higher value scrolls slower.

<Demo name="marquee/basic" />

## Directions

`direction` sets the scroll axis: `left` or `right` for horizontal, `up` or
`down` for vertical (vertical needs the root to have a height).
