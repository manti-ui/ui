# Button

The workhorse action. Five color variants plus a `link` style, three sizes, a
smooth press, and a loading state that keeps the button from collapsing.

<Demo name="button/variants" center />

## Variants

`variant` is the single color-and-emphasis dimension. `primary` (orange solid),
`secondary` (neutral soft), and `tertiary` (neutral ghost) form an emphasis
ladder; `danger` (red solid) is the semantic hue and `outline` is a neutral
bordered treatment. The Button-only `link` value renders as inline text. Any
custom variant string works too.
