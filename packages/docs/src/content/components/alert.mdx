# Alert

An inline, colored message with an optional icon, title, description, and dismiss
action. It defaults to `role="status"`, escalating to `role="alert"` for the
danger variant.

<Demo name="alert/variants" />

## Variants

`variant` carries the color and semantic meaning. It defaults to `secondary`; pass any
built-in variant (`primary`, `secondary`, `tertiary`, `danger`, `outline`) or a custom one.
Pass a `title` for the bold leading line and children for the description.

<Demo name="alert/variants" />

## Dismissible

Provide `onDismiss` to render a dismiss button. Drive the Alert's visibility
from state so it can be hidden — and brought back — on demand.
