# Slider

A draggable value selector backed by the Zag.js slider machine. The machine owns
pointer dragging, keyboard stepping, and the filled range; you set the bounds and
variant.

<Demo name="slider/basic" />

## Range and marks

Pass an array to `defaultValue` (or `value`) to get a two-thumb range slider, and
`marks` to render tick marks along the track.
