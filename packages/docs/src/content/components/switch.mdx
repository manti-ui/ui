# Switch

A smooth on/off control backed by the Zag.js switch machine. The machine owns
state, keyboard, and form behavior; toggle it with a click, the Space key, or a
controlled `checked` prop.

<Demo name="switch/basic" center />

## States

A Switch can start on with `defaultChecked`, sit off by default, or be
`disabled` to block interaction. Add a trailing label by passing children.

<Demo name="switch/basic" />

## Sizes

`size` offers a compact `sm` and the default `md`.

<Demo name="switch/sizes" />

## Variants

`variant` colors the control when it is on. Built-in variants are `primary`,
`secondary`, `tertiary`, `danger`, and `outline`.
