# Badge

A compact status or label chip. It comes in five variants, two sizes, and an
optional leading status dot.

<Demo name="badge/variants" center />

## Variants

`variant` is the single color-and-emphasis dimension. `primary`, `secondary`,
and `tertiary` form an emphasis ladder; `danger` is the semantic hue and
`outline` is a neutral bordered treatment. Any custom variant string works too.

<Demo name="badge/variants" />

## Dot

Set `dot` to add a small leading status dot, handy for presence and state.
