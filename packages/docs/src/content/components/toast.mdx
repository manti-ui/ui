# Toast

Toasts are created imperatively. `createToaster` returns a `toaster` store and a
`Toaster` host element: render `<Toaster />` once near your app root, then call
`toaster.create(...)`, `toaster.success(...)`, `toaster.error(...)`, and friends
from anywhere. Each type maps to a semantic variant and icon.
