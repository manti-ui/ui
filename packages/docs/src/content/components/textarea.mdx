# Textarea

A static multiline text input with label, helper text, error state, and optional
auto-resize. It wires up label association and `aria-describedby` /
`aria-invalid` automatically, and accepts native textarea attributes.

<Demo name="textarea/basic" />

## Sizes and states

`size` sets the control rhythm, `rows` sets the initial height, `error` flips the
field to the invalid state and replaces helper text, and native flags like
`disabled` work as expected.
