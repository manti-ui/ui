# Card

The signature pillowy surface ‚Äî the most recognizable Manti UI shape. Compose
it from `Card.Header`, `Card.Title`, `Card.Description`, `Card.Body`, and
`Card.Footer`.

## Anatomy

A Card is a slot container. Drop a `Card.Header` (with a `Card.Title` and
`Card.Description`), a `Card.Body` for the main content, and a `Card.Footer` for
actions.

<Demo name="card/basic" />

## Elevated & interactive

`elevated` swaps the border for a soft shadow, and `interactive` adds a smooth
hover-and-focus lift ‚Äî pair it with real semantics like a link, button, or
`onClick`.
