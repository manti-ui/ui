# Clipboard

A copy-to-clipboard field backed by the Zag.js clipboard machine. Press the
button to copy `value`; the icon flips to a check for a moment to confirm.

<Demo name="clipboard/basic" center />

## Basic

Pass the text to copy as `value` and an optional `label`. The `variant` colors the
copied confirmation, and `timeout` controls how long it lingers.
