# SegmentedControl

A single-select control rendered as a row of segments, with a pill that slides
to the active one. It is backed by the Zag.js radio-group machine, so it keeps
full keyboard support, roving focus, and form semantics — the machine measures
the selected segment and drives the indicator.

<Demo name="segmented-control/basic" center />

## Basic

Pass `items` as `{ value, label, disabled? }` and set an initial `defaultValue`
(or control it with `value` + `onValueChange`). The sliding pill fills with the
`variant` (primary by default) and the active segment's label flips to the
contrasting on-variant color.

<Demo name="segmented-control/basic" center />

## Sizes

`size` sets the segment height and padding: `sm`, `md` (default), or `lg`.
