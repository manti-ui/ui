# Design signature

Manti UI has one driving word: **smooth**. Surfaces feel soft and continuous,
interaction feels calm and predictable, and the whole system stays quiet so your
content can speak. The visual signature is a sleek, monochrome, cool-dark panel —
a single translucent surface lit from the top.

## Monochrome by default

Surfaces are **cool, near-neutral, and monochrome**. Dark is the hero: a deep
near-black with a faint cool cast. Light is a soft daylight that never goes stark
white. Emphasis is carried entirely by neutral light and dark — a top-lit rim, a
neutral focus ring, near-white solids on dark and near-black solids on light.

There are deliberately **no gradients** and **no colored brand accent**. The
interface reads as one quiet material. Color enters only through the semantic
[variants](/foundations/color-and-tones) — an orange primary; neutral secondary,
tertiary, and outline treatments; and a red danger — applied to badges, alerts,
and the like, never as decoration.

## The translucent panel

Floating surfaces — cards, tooltips, the field control, soft buttons, soft
badges, soft alerts, toggles, accordions, collapsibles — share one **translucent
panel** material:

- a light `backdrop-filter: blur() saturate()`,
- a hairline border,
- a top-lit rim that catches the light,
- and a soft, deep shadow.

This translucent panel is pervasive. Only **solid** fills (which gain nothing from a
blur) and the tiny form marks — checkbox, radio, switch — stay crisp.

You can give your own floating surfaces — menus, sheets, command palettes — the
exact same material with the `.manti-panel` helper. Where `backdrop-filter` is
unsupported, every panel surface falls back to an opaque token surface
automatically.

## Top-lit rim and ambient mesh

Two details create depth without a single gradient element. A **top-lit rim**
(`--manti-panel-highlight`) runs along the upper edge of each panel, as if a soft
light sat just above the screen. Behind every `.manti-app` or `body` sits an
**ambient mesh**: three faint glows — violet, magenta, and warm ember at very low
opacity (`--manti-ambient-1` … `--manti-ambient-3`) — as pure scene lighting. The
deep background reads with depth rather than flat black.

Retune the entire material from one place:

```css
.manti-app {
  --manti-panel-tint: oklch(0.2 0.008 280 / 0.55);
  --manti-panel-blur: blur(16px) saturate(160%);
  --manti-panel-border: oklch(1 0 0 / 0.09);
  --manti-panel-highlight: oklch(1 0 0 / 0.08);
}
```

## The smooth promise

Smooth is not only the look — it is the motion. Transitions ride a calm
ease-out (`--manti-ease-smooth`) by default, and the system honors
`prefers-reduced-motion` everywhere. See [Motion](/foundations/motion) for the
tiers and curves.
