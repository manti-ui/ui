# Swipe

Make any content swipeable, horizontally and/or vertically. Backed by the
framework-agnostic swipe core in `@manti-ui/folds`. By default the content
follows the pointer and springs back on release while `onSwipe` reports the
recognized direction.

<Demo name="swipe/basic" center />

## Swipe to dismiss

Constrain to one `axis` and remove the item in `onSwipe` to build a
swipe-to-dismiss stack — the component handles the gesture, your code removes the
card.
