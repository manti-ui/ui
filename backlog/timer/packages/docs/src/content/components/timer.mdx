# Timer

A count-up / count-down timer backed by the Zag.js timer machine. Choose which
segments to show with `parts`, and opt into start / pause / reset buttons with
`controls`.

<Demo name="timer/basic" center />

## Countdown

Set `countdown` and a `startMs` offset to count toward zero. `onComplete` fires
when it reaches it.
